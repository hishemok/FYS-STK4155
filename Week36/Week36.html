a) Expression for Ridge regression

Show that the optimal parameters $$\mathbf{\hat{\beta}}_{Ridge} = (\mathbf{X}^T\mathbf{X} + λ \mathbf{I})^{-1} \mathbf{X}^T\mathbf{y}$$ with $\mathbf{I}$ being a $p × p$ identity matrix.

The ordinary least squares result is $$\mathbf{\hat{\beta}}_{OLS} = (\mathbf{X}^T\mathbf{X})^{-1} \mathbf{X}^T\mathbf{y}$$

To show that the optimal parameters for Ridge regression are given by:

$$\mathbf{\hat{\beta}}_{Ridge} = (\mathbf{X}^T\mathbf{X} + λ \mathbf{I})^{-1} \mathbf{X}^T\mathbf{y}$$

we need to derive this from the Ridge regression cost function.

The cost function for Ridge regression is given by: $$ C(\mathbf{\beta}) = (\mathbf{y} - \mathbf{X}\mathbf{\beta})^T (\mathbf{y} - \mathbf{X}\mathbf{\beta}) + λ \mathbf{\beta}^T \mathbf{\beta} $$

Expand the Cost Function: $$ C(\mathbf{\beta}) = (\mathbf{y}^T - \mathbf{\beta}^T \mathbf{X}^T)(\mathbf{y} - \mathbf{X}\mathbf{\beta}) + λ \mathbf{\beta}^T \mathbf{\beta} $$ $$ C(\mathbf{\beta}) = \mathbf{y}^T \mathbf{y} - \mathbf{y}^T \mathbf{X} \mathbf{\beta} - \mathbf{\beta}^T \mathbf{X}^T \mathbf{y} + \mathbf{\beta}^T \mathbf{X}^T \mathbf{X} \mathbf{\beta} + λ \mathbf{\beta}^T \mathbf{\beta} $$

then combine the terms: $$ C(\mathbf{\beta}) = \mathbf{y}^T \mathbf{y} - 2 \mathbf{y}^T \mathbf{X} \mathbf{\beta} + \mathbf{\beta}^T (\mathbf{X}^T \mathbf{X} + λ \mathbf{I}) \mathbf{\beta} $$

Take the Gradient with Respect to $\mathbf{\beta}$ and set equal to zero for optimization: $$ \frac{\partial C(\mathbf{\beta})}{\partial \mathbf{\beta}} = -2 \mathbf{X}^T \mathbf{y} + 2 (\mathbf{X}^T \mathbf{X} + λ \mathbf{I}) \mathbf{\beta} = 0$$

$$ -2 \mathbf{X}^T \mathbf{y} + 2 (\mathbf{X}^T \mathbf{X} + λ \mathbf{I}) \mathbf{\beta} = 0 $$ $$ (\mathbf{X}^T \mathbf{X} + λ \mathbf{I}) \mathbf{\beta} = \mathbf{X}^T \mathbf{y} $$

Solve for $(\mathbf{\beta})$: $$ \mathbf{\beta} = (\mathbf{X}^T \mathbf{X} + λ \mathbf{I})^{-1} \mathbf{X}^T \mathbf{y} $$

Thus, we have shown that the optimal parameters for Ridge regression are:

$$\mathbf{\hat{\beta}}_{Ridge} = (\mathbf{X}^T\mathbf{X} + λ \mathbf{I})^{-1} \mathbf{X}^T\mathbf{y}$$

b) The singular value decomposition

OLS

The singular value decomposition of an $n × p$ matrix $\mathbf{X}$: $$\mathbf{X} = \mathbf{UΣV}^T$$ $\mathbf{U}$ and $\mathbf{V}$ are orthogonal matrices of dimensions $n × n$ and $p × p$, and $\mathbf{\Sigma}$ is an $n× p$ matrix which contains the singular values only.

First we show that you can write the OLS solutions in terms of the eigenvectors (the columns) of the orthogonal matrix $\mathbf{U}$ as $$\mathbf{\tilde{y}}_{OLS} =\mathbf{X\beta} = \sum_{j=0}^{p-1} \mathbf{u_j} \mathbf{u_j}^T\mathbf{y}$$

The OLS solution for $\mathbf{\beta}$ is given by: $$\mathbf{\hat{\beta}} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$

First we can find $\mathbf{X^TX}$: $$\mathbf{X}^T \mathbf{X} = (\mathbf{UΣV}^T)^T (\mathbf{UΣV}^T) = \mathbf{VΣ}^T \mathbf{U}^T \mathbf{UΣV^T} = \mathbf{VΣ}^T \mathbf{ΣV^T}$$ Since $\mathbf{U}$ is orthogonal, $\mathbf{U}^T \mathbf{U} = \mathbf{I}$.

Inverse of $\mathbf{X}^T \mathbf{X}$: $$ (\mathbf{X}^T \mathbf{X})^{-1} = (\mathbf{VΣ}^T \mathbf{ΣV^T})^{-1} = \mathbf{V}^{-T} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{V}^{-1} = \mathbf{V} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{V}^{T}$$ Since $\mathbf{V}$ is orthogonal, $\mathbf{V}^{-1} = \mathbf{V}^T$.

Substitute Back into OLS Solution: $$\mathbf{\hat{\beta}} = \mathbf{V} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{V}^{T} \mathbf{X}^T \mathbf{y}$$ $$\mathbf{\hat{\beta}} = \mathbf{V} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{V}^{T} \mathbf{V\Sigma^TU^T} \mathbf{y}$$ $$\mathbf{\hat{\beta}} = \mathbf{V} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{Σ}^T \mathbf{U}^T \mathbf{y}$$

Compute $\mathbf{\tilde{y}}_{OLS}$: $$\mathbf{\tilde{y}}_{OLS} = \mathbf{X\hat{\beta}} = \mathbf{UΣV}^T \mathbf{V} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{Σ}^T \mathbf{U}^T \mathbf{y}$$ $$\mathbf{\tilde{y}}_{OLS} = \mathbf{UΣ} (\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{Σ}^T \mathbf{U}^T \mathbf{y}$$ And since $(\mathbf{\Sigma^T\Sigma}) = diag(\sigma_0^2,\sigma_1^2,...,σ_i^2)$

and $(\mathbf{\Sigma^T\Sigma})^{-1} = diag(1/\sigma_0^2,1/\sigma_1^2,...,1/σ_i^2)$ then $\mathbf{\Sigma}(\mathbf{Σ}^T \mathbf{Σ})^{-1} \mathbf{Σ}^T= \mathbf{I}$, so:

$$\mathbf{\tilde{y}}_{OLS} = \mathbf{U} \mathbf{U}^T \mathbf{y}$$

Express in Terms of Eigenvectors: Since $\mathbf{U}$ is orthogonal, it can be written as a sum of its eigenvectors: $$\mathbf{U} \mathbf{U}^T = \sum_{j=0}^{p-1} \mathbf{u_j} \mathbf{u_j}^T$$

Final Expression: $$\mathbf{\tilde{y}}_{OLS} = \sum_{j=0}^{p-1} \mathbf{u_j} \mathbf{u_j}^T \mathbf{y}$$

Ridge

For Ridge regression, show that the corresponding equation is $$\mathbf{\tilde{y}}_{Ridge} = \mathbf{X\beta}_{Ridge} = \mathbf{UΣ V}^T(\mathbf{VΣ^2 V^T } + λ \mathbf{I})^{-1} (\mathbf{U Σ V^T})^T\mathbf{y} = \sum_{j=1}^{p-1}\mathbf{u_ju_j^T} \frac{σ_j^2 }{\sigma_j^2 +\lambda} \mathbf{y}$$

First we start with: $$\mathbf{UΣ V}^T(\mathbf{VΣ^2 V^T } + λ \mathbf{I})^{-1} (\mathbf{U Σ V^T})^T\mathbf{y}$$ and re-write to: $$\mathbf{UΣ V}^TV(\mathbf{Σ^2 } + λ \mathbf{I})^{-1} V^T(\mathbf{U Σ V^T})^T\mathbf{y}$$

Then we can further simplify: $$\mathbf{U\Sigma(\Sigma^2+λ I)^{-1}\Sigma^TU^Ty}$$ Now we can take a look at $\mathbf{\Sigma}(\mathbf{\Sigma}^2 +λ \mathbf{I})^{-1} \Sigma^T$ element by element, $$\mathbf{\Sigma}(\mathbf{\Sigma}^2 +λ \mathbf{I})^{-1} \Sigma^T = (\frac{\sigma_0^2}{\sigma_0^2 + λ} , \frac{\sigma_1^2}{\sigma_1^2 + λ}, \frac{\sigma_i^2}{\sigma_i^2 + λ})$$ Final expression: $$\mathbf{\tilde{y}}_{Ridge} = \mathbf{UΣ V}^TV(\mathbf{Σ^2 } + λ \mathbf{I})^{-1} V^T(\mathbf{U Σ V^T})^T\mathbf{y} = \sum_{j=1}^{p-1}\mathbf{u_j}\mathbf{u_j}^T\frac{\sigma_j^2}{\sigma_j^2 + λ} \mathbf{y}$$

def scaling(X,y):
    xmean = np.mean(X, axis=0)
    xstd = np.std(X, axis=0)
    xstd[xstd == 0] = 1 #avoid division by zero
    ymean = np.mean(y)
    ystd = np.std(y)
    X_scaled = (X - xmean)/xstd
    y_scaled = (y - ymean)/ystd
    X_scaled[:,0] = 1
    return X_scaled, y_scaled, xmean, xstd, ymean, ystd
